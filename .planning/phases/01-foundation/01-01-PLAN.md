# Plan 01-01: Backend Foundation

## Goal
Set up the .NET 8 solution with Clean Architecture and PostgreSQL configuration.

## Tasks
- [ ] Task 1: Initialize .NET Solution
  - Create `MoneyFlow.sln`.
  - Create projects: `MoneyFlow.Domain`, `MoneyFlow.Application`, `MoneyFlow.Infrastructure`, `MoneyFlow.API`.
  - Add project references.
- [ ] Task 2: Configure Domain & Application Layers
  - Set up BaseEntity and common domain types.
  - Initialize MediatR and FluentValidation in Application.
- [ ] Task 3: Infrastructure & PostgreSQL Setup
  - Install EF Core PostgreSQL provider.
  - Set up `ApplicationDbContext`.
  - Configure connection string in `appsettings.json`.
- [ ] Task 4: API Setup
  - Configure Serilog.
  - Set up Global Exception Handling middleware.
  - Configure Swagger with JWT support.
  - Implement API Versioning.

## Verification
- [ ] `dotnet build` succeeds.
- [ ] Swagger UI is accessible at `/swagger`.
- [ ] Connection to PostgreSQL is successful (verified via health check or migration).
