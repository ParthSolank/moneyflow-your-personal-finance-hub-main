# Plan 05-01: Bank Statement AI Ingestion

## Goal
Implement a robust bank statement parsing engine to automate transaction entry from PDF statements.

## Tasks
- [ ] Task 1: PDF Parsing Library
  - Add `PdfPig` to `Infrastructure` project.
- [ ] Task 2: Statement Parsing Engine
  - Create `IPdfParserService` in `Application`.
  - Implement `PdfParserService` in `Infrastructure`.
  - Add logic for basic transaction row extraction (Date, Description, Amount).
- [ ] Task 3: Import Features (CQRS)
  - Implement `ParseStatementCommand` (returns preview data).
  - Implement `ConfirmImportCommand` (saves transactions to DB).
- [ ] Task 4: API Controllers
  - Create `ImportController` with `MultipartFormData` support for file uploads.
- [ ] Task 5: Frontend Integration
  - Update `ImportStatement.tsx` to handle file uploads.
  - Implement a preview table where users can review and edit parsed data before saving.
  - Add account selection for the import.

## Verification
- [ ] PDF file is successfully uploaded and read.
* [ ] Transactions are correctly extracted from sample statement layouts.
- [ ] User can preview and modify transactions before final import.
- [ ] Imported transactions appear in the Transactions page and update account balances.
